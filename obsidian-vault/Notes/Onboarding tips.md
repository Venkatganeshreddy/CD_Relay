# Onboarding tips

Human-authored note (not generated). Edit or add files under `Notes/`, then run `scripts/import_obsidian.cjs` to sync into Supabase so Concierge can answer from them.

- New L1s: submit your first daily report the same day you join; the 5:30 PM banner is your cue.
- Blocked tasks need a reason — Sentry escalates anything blocked >72h.
- Ask [[Concierge]] "what's the Daily Report SOP?" if unsure of the format.
